# 3D-Printable Components

All parts are available as STL files (ready to print) and IPT CAD files (editable) in the [hardware repository](https://github.com/PublicQuantumNetwork/hardware).

## Base

A cylinder with a **1-inch diameter**. An adhesive magnet is bonded to the bottom so the base can be repositioned on any magnetic surface. Posts friction-fit into the top.

## Posts

**½ inch diameter, 4 inches tall** (an 11.5 cm variant is also provided). The post slides into the base with enough pressure applied. All mounts attach to the post via a pressure-fit hole.

## Mounts

Five mount types accommodate different optical components. Each has a hole that pressure-fits onto a post.

| Mount | Fits |
|-------|------|
| Laser Mount | 11 mm diameter laser |
| Lens Mount | 1-inch diameter Fresnel lens; clamping filament holds the lens in place |
| Retardation / Polarizer Mount | Films < 1 mm thick (polarizing film, wave plates) |
| Beam Splitter Mount | 1 mm thick beam splitter |
| Mirror Mount | Mirror with adhesive backing |

## Wheel

Attaches to the waveplate (retardation plate) and allows manual rotation of the polarization basis. Rotating the wheel changes the superposition state measured during an experiment.
